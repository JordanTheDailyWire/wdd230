Hick's Law

Amazon

Amazon

I would say Amazon fits into a hick's law. Where the user is given more options to chosen from which makes the user take longer to decide on that item. The more the user clicks on an item to view the more options is given to the user through that item. It will take the user a while to shop on Amazon because they want to find the best affordable product on Amazon. As more products are being released, the more that Amazon will increase there options.
